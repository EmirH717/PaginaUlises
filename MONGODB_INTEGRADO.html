🚀 Guía de Instalación (5 Pasos)

1

Instalar Node.js

Descarga de https://nodejs.org/ (versión LTS recomendada)

Verifica en terminal:

node --version npm --version
2

Instalar Dependencias

En la terminal, navega a tu carpeta y ejecuta:

cd path/to/PaginaUlises2 npm install

Esto instalará Express, Mongoose, JWT y más (~200MB)

3

Configurar MongoDB Atlas

Lee el archivo MONGODB_SETUP.md que incluye:

  • Crear cluster en Atlas
  • Crear usuario de BD
  • Obtener connection string
  • Actualizar .env

✓ Ya incluye configuración de demostración

4

Inicializar Base de Datos

Crea productos y usuario admin automáticamente:

node scripts/inicializar-bd.js

Deberías ver:

✅ Conectado a MongoDB Atlas ✅ 6 productos insertados ✅ Usuario administrador creado
5

Iniciar Servidor

En la terminal ejecuta:

npm start

Verás:

✅ Conectado a MongoDB Atlas 🚀 Servidor ejecutándose en puerto 5000

¡El servidor está listo!

🐛 Solución de Problemas

❌ "npm: command not found"

Node.js no está instalado. Descargalo desde nodejs.org y reinicia la terminal.

❌ "Cannot find module 'mongoose'"

Ejecuta npm install en la carpeta del proyecto.

❌ "Connection refused"

Verifica que MongoDB Atlas esté disponible y tu IP esté en whitelist.

❌ "Puerto 5000 en uso"

Cambia PORT en .env a otro número (5001, 5002, etc)

❌ "MONGODB_URI undefined"

Verifica que .env exista en la raíz y contenga MONGODB_URI.

❌ Productos no cargan

Asegúrate de ejecutar primero: node scripts/inicializar-bd.js
