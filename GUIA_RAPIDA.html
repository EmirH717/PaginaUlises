⚙️ Cómo Personalizar

Cambiar nombre de la tienda

Edita "FashionStyle" en cada archivo HTML, especialmente en el logo y títulos.

Cambiar colores

En css/estilos.css, modifica las variables CSS:

:root {
    --color-primario: #6c3483;
    --color-secundario: #9b59b6;
    --color-acento: #e74c3c;
    ...
}

Agregar productos

Ve al panel admin y usa la opción "Agregar Producto" para crear nuevos artículos.

Agregar imágenes

Coloca las imágenes en la carpeta /imagenes/ y usa el nombre en los productos.

💡 Tips Útiles

  • Limpiar datos: Abre la consola (F12), ve a Storage > LocalStorage y elimina los datos.
  • Ver datos guardados: Abre la consola, ve a Storage > LocalStorage para inspeccionar.
  • Responsive: Prueba el diseño redimensionando la ventana o usando dispositivos móviles.
  • Desarrollo local: Puedes usar Live Server de VS Code para mejor experiencia.
